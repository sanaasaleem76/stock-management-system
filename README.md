# 📦 Stock Management System

## Overview

The Stock Management System is a software application developed as a semester project to simplify inventory management. The system allows users to manage product records, monitor stock levels, and perform essential inventory operations through an organized and user-friendly interface.

---

## Features

- Add new products
- Update product information
- Delete product records
- View inventory details
- Manage stock quantities
- Search products
- User-friendly interface

---

## Technologies Used

## How to Run

1. Clone the repository

```bash
git clone https://github.com/sanaasaleem76/stock-management-system.git
```

2. Install dependencies

```bash
flutter pub get
```

3. Run the application

```bash
flutter run
```
